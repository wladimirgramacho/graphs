Instruções de compilação:

g++ -o main main.cpp

Instruções de execução:
./main

OBS: é necessário que o programa 'gnuplot' esteja instalado.
Para instação no Ubuntu ou em outras distribuições baseadas no Debian, usar
o comando abaixo:

sudo apt-get install gnuplot

para mais informações:
http://www.gnuplot.info/
